# Instagram Web UI Replica 📸

A highly detailed, fully responsive frontend clone of the Instagram web application built from scratch using vanilla web technologies.

## ✨ Features
- **Responsive Design**: Adapts perfectly between Desktop sidebar view and Mobile top/bottom nav views.
- **Dark/Light Mode**: Seamless theme toggling that saves user preference using `localStorage`.
- **Interactive UI**: Working "Like" buttons with dynamic counters, and functioning comment submission boxes that update the DOM instantly.
- **Complete Navigation**: Fully wired UI containing 6 separate functional views:
  - Feed
  - Search / Explore
  - Reels (Vertical video scroll)
  - Messages (Chat UI layout)
  - Notifications
  - Profile
- **"Create" Modal**: Working popup overlay to simulate new post creation across all pages.

## 🛠️ Tech Stack
- **HTML5**: Semantic structuring
- **CSS3**: Advanced CSS Custom Properties (Variables), Flexbox, and CSS Grid layouts
- **Vanilla JavaScript**: DOM manipulation and state handling (No frameworks used)
- **Phosphor Icons**: For beautiful, modern iconography

## 🚀 How to Run Locally
1. Clone this repository: `git clone https://github.com/muhammadshahzaib585/FentixTech-Instagram-UI-Clone.git`
2. Open the folder and launch `feed.html` in your browser. (Or use a local server like Live Server or `npx serve`).
